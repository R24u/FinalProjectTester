# Final Project Tester

write Test Scenarios for the following tests:

## 1. Java Code Testing with JUnit

- Test the assigned Java code using **JUnit**
- Create test cases until **coverage exceeds 90%**
- All groups must test both assigned Java programs

## 2. Web UI Testing with Katalon Studio

All groups must test the following 3 websites:

- [XYZ Bank - AngularJS Banking Project](https://www.globalsqa.com/angularJs-protractor/BankingProject)
- [Katalon CMS Demo](https://cms.demo.katalon.com)
- [CURA Healthcare Service](https://katalon-demo-cura.herokuapp.com)

## 3. API Testing with Postman and Katalon Studio
